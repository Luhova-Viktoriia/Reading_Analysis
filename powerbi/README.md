# Reading Analytics Dashboard | Power BI

## Overview

This project demonstrates the development of an end-to-end Power BI reporting solution based on a relational PostgreSQL database containing personal reading data.

The solution combines data modeling, DAX calculations and interactive visualizations to analyze reading behavior, book series progression, and genre distribution through three analytical dashboards.

## Dashboards

### 1. Reading Behavior Analytics
Provides a high-level overview of reading activity through KPI cards, monthly trends, diversity metrics, author insights and publication year analysis.

Key techniques:
- DAX measures
- KPI calculations
- Time-based aggregations
- Ratio and percentage measures
- Interactive filtering

### 2. Reading Series Progress & Structure Analytics
Analyzes reading progress across book series, completion status and relationships between genres, subgenres and series.

Key techniques:
- Hierarchical drill-down
- Calculated columns
- Progress classification
- Conditional formatting
- Dynamic measures

### 3. Top 30 Books – Genre & Subgenre Curation Analytics
Visualizes the highest-rated books of 2025 using a treemap with custom genre classification and interactive highlighting.

Key techniques:
- Custom business logic
- Genre hierarchy
- Ranking measures
- Treemap visualization
- Interactive cross-highlighting
